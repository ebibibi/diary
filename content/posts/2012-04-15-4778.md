自分用メモ。自分が管理しているサーバー上でwordpressのブログを増やす場合の手順。

- URLを決める
- cd /etc/wordpress
- cp config-ebi.dyndns.biz\_xxxx.php config-ebi.dyndns.biz\_.php
- ln -s config-ebi.dyndns.biz\_.php config-ebi.dyndns.biz\_\_wp-admin.php
- emacs config-ebi.dyndns.biz\_.php

- define('DB\_NAME', 'wp\_');

- chown www-data
- chgrp www-data
- mysql -p -u root

- mysql> create database wp\_;
- mysql> grant all on wp\_.
- mysql> flush privileges;
- mysql> quit

- ln -s /usr/share/wordpress /var/www/html/

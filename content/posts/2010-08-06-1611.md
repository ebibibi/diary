今日は初台でMAC OS Xの試験を2つ受けてきました。

- MAC OS X Server Essentials
- MAC OS X Support Essentials

難易度的にはMCPと同じか若干簡単だったように思います。まぁ、別に難しくないですが、でもきちんと範囲を勉強して、頭に数字等いれておかないと絶対に合格しない試験でした。試験にどのような問題がでるのかを把握していないと厳しいでしょうけど、それさえわかっていれば(努力すれば)誰でも簡単に合格する試験だと思いました。

今回は会社に取れと言われてとったわけですが、こういうものはそろそろ「いかに短い学習時間で効率的に合格するか」という領域になってきたように思います。あんまり時間かけ過ぎてもそれに見合ったリターンがあるとも思えないし。

まぁ、なんにしても、持ち駒(はったりの道具)が増えたのはよいことです。

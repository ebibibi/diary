出発前には何かと不安の多かった今回の旅(出張)も無事に日程をこなして、帰りの飛行機の中でこの日記を書いています。帰りはここまでのところほとんど揺れずに快適です！ただ、機内サービスの機械が不調で映画や音楽等が聞けない状態だったりしますが、私は元から本とiPadを使って勉強したり、文章を書いたりする予定だったのでセーフ。人によってはずいぶんと暇をしているところだと思います。

{{flickr '7703549686'}}

delta航空の端末はlinuxベースで動いてました。何度も再起動をするたびに画面が見えて面白かったです。どうやら起動してからコンテンツをローカルに全部キャッシュし、それからシステムが使用可能になる仕組みのようで、その最新のコンテンツを保持しているファイルサーバー的なものの調子が悪く、コンテンツのダウンロードが中々終わらずにシステムが立ち上がらないようです。中にはごく少数きちんとシステムがたちあがっている席もあったりします。システムに詳しいスタッフが機内にいるはずもなく、「なんだかおかしいからもう一度再起動をするね。30~40分くらいかかるよ」って言いながら再起動を繰り返すものだから、ますますシステムが立ち上がらないように見えます。残念。

今回はパートナー企業向けのWindows Server 2012の新機能の概要に関してのイベントでした。内容に関してはcommunity dayと同じ程度の内容、レベルでした。わざわざシアトルまでこなくても……という内容ではあったわけですが、実際の製品担当のMicrosoft本社の人から直接話を聞くことができ、特にWindows Server 2012に対してみんな自信をもってやっていることを肌で感じることができたのは良かったです。個別の内容に関しては、ためしつつ、まとめていこうと思います。

また、日本からは3企業、3人の参加だったのですが、みんな英語がそれほどできるわけでもなかったこともあり色々と同業他社の話を聞けたのは非常に面白かったです。社員数や規模はずいぶんと違うところだったので、規模に応じた苦労とか、どこでも共通する苦労とか……。「やっぱりそうですよねー」という話が多かったです。Windows中心の企業インフラ構築……となると、結局どこでも同じようなことをやっていて企業間の差異というのは出づらいので、こういうところでつながりを作っておくと職に困ることは無いんじゃないかな、という気がしました(笑

先日の日記にも書きましたが、個人的な一番の関心事はやっぱり「英語」でした。今回の良かったところは以下。

- 普通に買い物をしたり、食事を注文したり、ホテルでフロントの人と会話等をするのには不自由を感じない。
- セッションでも、人によってはほぼ全部ヒアリング出来た。
- まがりなりにも、日本語が全くしゃべれない人と、30分、1時間単位でそれなりに楽しく会話ができた。

逆によくなかったのは以下

- 不自由はしなかったけれども、全部聞き取れたり、スラスラとしゃべれたわけでは無い。何度も聞き直したり、ゆっくりしゃべってくれってたのんだり。特にしゃべる時の文法は本当にひどい。あとになってゆっくり考えるときちんと組み立てられることも多いんだけど。
- セッションで、数人がしゃべっていることはほとんど聞き取れなかった。
- セッション中に出た質問はほとんど聞き取れなかった。
- 超簡単な単語でも発音の問題で聞き取ってもらえないことがあってとまどった。(非ネイティブの人にworkも通じなかった)
- やっぱりそんなに積極的に話しかけられなかった。質問できなかった。
- ネイティブ同士の会話はほとんどなんにも聞き取れない!!本当に英語だよね?というレベル…。

過去の出張に比べると格段に英語力が伸びていることは実感できました。流石に英語学習を何年もゆるりと継続しているだけの成果は出てきてます。これに気を良くして学習は継続できそうです。ただ、人によって全然聞き取れなかったのがかなりショックでした。Windows Serverまわりであれば単語もほとんど知ってるだろうし、大丈夫だろうと思ってたので…。英語力学習者用のくっきりはっきり英語だけではなく、もっとネイティブ同士の会話とか、くせのある英語などの聞き取り練習をしなければと思います。

そう思って探してみたら、[Listen and Write - Dictation](http://www.listen-and-write.com/)というサービスがあり、簡単なものから難しいものまでレベル付けされているものをディクテーションできるみたいなのでこれを学習に取り入れてみようかなと思います。また、オンライン英会話もやってみようかとも思いますが……、こちらはどこがいいんだろうとか、継続すると高いよなとか、子供もいるから時間の確保がむずかしくて、朝はやくってのも眠いよな、とか、そういう事が気になってしまってイマイチはじめる気になれてません。でも、言っててもしかたないので、まずはあちこちの無料レッスンを受けてみようかなと思います。

MacOS Xを使う上でずっとKeePassからのコピペができないことが大きなハードルになってました。おそらくmonoで動いているアプリケーション上のクリップボードとの連携がうまく行かないのだろうと思いつつ、ずっとほったらかしにしていたのですが、そろそろMac上でまたiPhoneアプリを開発したく、そのためにはこの障害を取り除いておかなくてはいけません。

というわけで、色々挑戦してみます。

- [Downloads - KeePass](http://keepass.info/download.html)
- [Setup - KeePass](http://keepass.info/help/v2/setup.html#mono)

とりあえず

```
If you want to use auto-type on Linux / Mac OS X / BSD / etc., you additionally need the xdotoolpackage.
```

ということなので、xdotoolpackageを使えば目的のことはできそうなのですが。

- [xdotool - fake keyboard/mouse input, window management, and more :: semicomplete.com - Jordan Sissel](http://www.semicomplete.com/projects/xdotool/)

とりあえず、インストールしてみる。

がやはり動かない。ググってみると、少なくとも2009年の段階ではそもそもMacOS上のmonoとのクリップボード経由でのやり取り自体できないという情報があちこちで目につく。

- [\[Mono-osx\] Copy & Paste Issue in WinForms](http://lists.ximian.com/pipermail/mono-osx/2009-March/002176.html)

うーん。やっぱりまだできないんだろうか………。今日もこうしてまた挫折するのでした。何回目の挫折か忘れました。いいよもう、Choromeにパスワード覚えてもらうよ………。

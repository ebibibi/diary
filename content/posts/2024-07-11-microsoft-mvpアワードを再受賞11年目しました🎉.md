Microsoftからコミュニティ活動への貢献を表彰され、Microsoft MVPアワードを受賞させてもらいました。

https://mvp.microsoft.com/ja-JP/mvp/profile/959007f9-3c9a-e411-93f2-9cb65495d3c4

2014年から連続受賞させてもらっており、今回で11回目の受賞となりました。

10年以上もやっていると私が取り組んでいることも色々と変化しておりまして、下記のような変遷をたどっております。

- (2014)Microsoft MVP for Windows Expert-IT Pro

- (2015)Microsoft MVP for System Center Cloud and Datacenter Management

- (2016-2018)Microsoft MVP for Cloud and Datacenter Management

- (2019-2023)Microsoft MVP for Microsoft Azure

- (2024)Microsoft MVP for Azure Hybrid & Migrate

- (2024)Microsoft MVP for Windows Server

今年は2つのカテゴリでの受賞となりました。

私自身が色々と変遷しているということもありますが、これはMicrosoftさん側自体の変遷でもあります。

こう見ると、M365系やDeviceManagement系やPowerplatform系のアワードも1回くらい受賞しておかないと実際に取り組んでいる事との受賞バランスが悪い気もしてきますね。

何はともあれ、今年も受賞できてよかったです。

でも、やってることは下記のくらいです。

1. 毎月のハイブリッドクラウド研究会の主催、コミュニティ運営、イベント運営、司会、配信、たまに登壇

3. Youtubeチャンネルでの情報発信(動画)

5. noteでの情報発信(文字)

7. Discordでのコミュニティ運営

なんとなく、今の感覚だと1と2だけきちんと継続していれば継続受賞できそうな気もしています。簡単?とはいえ、オンラインとはいえ毎月イベントを開催するのは当初はかなり大変ですよね。私はもう力の抜き方も分かってきたし、毎月開催は楽なものなんですけどね。

というわけで、引き続き情報発信していきたいと思いますので、皆さんよろしくお願いします。

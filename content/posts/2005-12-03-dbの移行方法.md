- MySQLの移行方法 MySQLのDBの移行方法が確立できました。 以下の2コマンドで簡単に移行できます。
    
    ```
    移行元：mysqldump -u root -p --default-character-set=binary xoopsdb > xoopsdb
    移行先：C:\Program Files\MySQL\MySQL Server 4.1\bin>mysql xoopsdb < xoopsdb -u root -p --default-character-set=binary
    ```
    
    ただしこれだけだと、(おそらく文字コードの違いで)情報の追加、修正時に文字化けがおきてしまうため、おそらく中間で文字コード変換が必要になると思われます。後日テスト予定。

- weblog, bwikiなどMySQLにデータが存在しないものの移行方法 単純にtarで固めてモジュールフォルダに入れてしまえば動きました。
    
    ```
    移行元：tar cvf xxxx.tar /xxxx
    移行先：上記ファイルを展開してc:\program files\apache group\apache2\htdoc\module以下にコピー。
    ```
    
    ただし、こちらも追加、修正分が文字化けしてしまうため、文字コード変換が必要な模様。

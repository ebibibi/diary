会社の社内システムでMediawikiを使おうとして今色々いじっているのですが、早速インストール直後に「管理者としてログインできない」という問題ではまりました。

インストール中のメニューにはきちんと管理者ユーザー名を入力する箇所があるので、それに入力しているのですが、インストール後にそのユーザー名、パスワードではログインできないし、MySQLのDBを覗いてみてもテーブルにそんなユーザーはできていないし・・・。

色々調べたところ以下のMLのやり取りにて回避方法が書かれていました。

http://lists.wikimedia.org/mailman/htdig/mediawiki-l/2006-June/012543.html

```
INSERT INTO user_groups (ug_user, ug_group) VALUES ('1', 'bureaucrat');
```

```
INSERT INTO user_groups (ug_user, ug_group) VALUES ('1', 'sysop');
```

こんな感じで直接SQL文にて権限を与えてしまえばよいとのこと。確かにこれでうまく行きましたが、やられた感がぬぐえません・・・。

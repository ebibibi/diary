#### [ダウンロードの詳細 : Exchange Server ExBPA](http://www.microsoft.com/downloads/details.aspx?FamilyID=DBAB201F-4BEE-4943-AC22-E2DDBD258DF3&displaylang=ja)

Exchange Server用のベストプラクティスアナライザーです。いろいろとサーバー構成の問題点等を洗い出してくれます。

#### [Files: Exchange 2007 Mailbox Server Role Storage Requirements Calculator spreadsheet](http://msexchangeteam.com/files/12/attachments/entry438481.aspx)

Exchangeチームが提供しているExchange Server 2007用のメールボックスサーバー用サイジングツール(Excel)です。

#### [Files: Exchange 2007 Mailbox Server Role Storage Requirements Calculator spreadsheet](http://msexchangeteam.com/files/12/attachments/entry438481.aspx)

Exchangeチームが提供しているExchange Server 2007用のトランザクションログ実測ツールです。

#### PFDavAdmin [Exchangeの管理を効率化する魔法のツール：ITpro](http://itpro.nikkeibp.co.jp/free/NT/WinColumn/20040630/1/)

Exchange Serverのパブリックフォルダのプロパティ設定やアクセス権設定等を行えるツールです。

#### [Download details: Microsoft Exchange Server Inter-Organization Replication Tool](http://www.microsoft.com/downloads/details.aspx?FamilyId=E7A951D7-1559-4F8F-B400-488B0C52430E&displaylang=en)

異なる組織(フォレスト)間でパブリックフォルダの複製やFree/Busy情報の複製を行えるツールです。

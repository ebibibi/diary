Mac OS X Lionにnokogiriを入れようとしたら入りませんでした。

```
MacBook-Pro:bin ebibibi$ sudo gem install nokogiri
```

```
Password:
```

```
Building native extensions.  This could take a while...
```

```
ERROR:  Error installing nokogiri:
```

```
	ERROR: Failed to build gem native extension.
```

ググるとXcodeを入れるといいよ、って話ばっかりだったのでXcodeの最新版を入れましたが、症状変わらず。結構困っていたのですが、結局XCodeがAppStoreで管理しやすいようにディレクトリ構成が変更されたのが影響していたようです。

- [Xcode 4.3 または CLI tools を入れた話 , デブサミ2012 の写真 - HsbtDiary(2012-02-18)](http://www.hsbt.org/diary/20120218.html#p01)

上記の記事をみて[Downloads](https://developer.apple.com/downloads/index.action#)からCommand Line Tools for Xcodeを導入したところ、インストールに成功しました。

```
MacBook-Pro:bin ebibibi$ sudo gem install nokogiri
```

```
Building native extensions.  This could take a while...
```

```
Successfully installed nokogiri-1.5.0
```

```
1 gem installed
```

いや、よかった。なんだかどのプラットフォームでもnokogiriのインストールの度に悩まされている気がします。

```
Notice [PHP]: Only variable references should be returned by reference in file xxxxxxx
```

といった感じで、多数の警告メッセージが表示されています。 Xoopsの公式サイトで検索してみたところ以下の回答が。

```
ユーザ・ログインはできますか？
そして、管理画面は開けますか？
管理画面が開けるようでしたら、[SYSTEM ADMIN]-[一般設定]-[一般設定]の[デバッグモードを有効にする]欄をオフに切り替えてください。
```

途中以下のメッセージが表示されたので、指示通りに修正する。

```
注意：ファイルC:/Program Files/Apache Group/Apache2/htdocs/install/がサーバ上に存在します。インストール完了後は必ず削除してください。
注意：ファイルC:/Program Files/Apache Group/Apache2/htdocs/mainfile.phpへの書き込みが可能となっています。このファイルのパーミッション設定を変更してください
```

書き込みを禁止するには、Windowsの場合、読み取り専用属性をつければよいようだ。

以上で一応インストールは完了。長かったです。

hikiのrefererプラグインを導入したのですが、除外URL設定がうまく動作していなかったので修正しました。

referer.rbの15行目からの部分を以下のように変更しました。

```
raise if /HEAD/i =~ @cgi.request_method || ! @cgi.referer ||
```

```
#                   /^https?/ !~ @cgi.referer || omit_url
```

```
                    /^https?/ !~ @cgi.referer || omit_url != nil
```

これで意図したとおりに動いている模様です。
